# Лабораторная работа №1: Первичное исследование и оценка качества данных

## Описание проекта
Первичный анализ датасета **blink.csv** в рамках дисциплины «Интеллектуальный анализ данных».

**Предметная область:** датасет содержит логи регистрации сигналов в системе позиционирования, где фиксируется взаимодействие якорей (anchor) и меток (tag) с временными метками и служебными параметрами. Данные позволяют выявлять аномалии в работе оборудования, анализировать последовательность событий и исследовать поведение системы во времени.  
**Цель лабораторной:** аудит качества данных, построение паспорта датасета, выявление проблем.

## Структура проекта
- `notebooks/01_data_understanding.ipynb` — основной Jupyter Notebook с анализом  
- `report/quality_report.md` — текстовый отчёт с выводами и планом улучшения  
- `data/` — датасет blink.csv
